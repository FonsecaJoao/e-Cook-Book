e-CookBook – store your recipes just the way you want to :)

Installation guide

  1. Download e-CookBook.
  2. Put e-CookBook folder on local or remote server.
  3. Open folder link in your browser.
  4. That's it! You're all set to use e-CookBook – go ahead and add the first recipe!

Requirements

  1. Server (local or external).
  2. PHP v.5x or higher with enable SQLite 3.
  3. Browser with JavaScript enabled.
