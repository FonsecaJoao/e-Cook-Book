e-Cook-Book helps you to keep recipes exactly like you want :)

How to install

  1. Download 'e-Cook-Book' to your computer.
  2. Put folder 'e-Cook-Book' on server - it can be your local server on your computer or it can be an external server.
  3. Open link to this folder on your browser.
  4. Now e-Cook-Book is ready to use :)

Requirements

  1. Server (local or external) with PHP v.5x or higher.
  2. Browser with enabled JavaScript.
